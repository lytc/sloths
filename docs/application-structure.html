Application Structure

Classic Style Application

Single Module Application

Normally, the application structure look like this:

\root-directory
    \application
        \config
        \models
        \routes
            index.php
        \views
            \helpers
        MyApplication.php
    \public
        index.php
    composer.json
public/index.php

Modular Application

Normally, the application structure look like this:

\root-directory
    \application
        \config
        \models
        \modules
            \content
                \routes
                \views
                ContentApplication.php
            \admin
                \routes
                \views
                AdminApplication.php
    \public
        index.php
    composer.json
public/index.php
